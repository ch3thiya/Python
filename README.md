# Python Scripts Repository

A collection of some noteworthy Python scripts I wrote while learning Python.

## Projects

1. **Amazon Price Checker**: A script to track price changes of Amazon products.
2. **Authentication**: A simple login system with username/password authentication.
3. **Blackjack Game**: A console-based Blackjack game with basic gameplay and dealer interactions.
4. **Bulk Mail Writer**: An automated script to write bulk personalized emails.
5. **Coffee Machine**: A simulation of a coffee machine with options for different types of coffee.
6. **Ping Pong Game**: A fun Ping Pong game.
7. **Pomodoro**: A time management tool following the Pomodoro Technique.
8. **Quiz App**: An interactive quiz app that uses APIs and presents questions to users.
9. **Snake Game**: A classic Snake game.
10. **Spotify Playlist Generator**: A tool to generate Spotify playlists based on the user's required period.
11. **Turtle Crossing Game**: A fun game built with the Turtle library.
